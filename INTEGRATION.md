# Platform Integration Guide

This guide explains how to integrate external systems (e.g. `paisawin.online`) with our tracking platform via Postback API.

## Postback API Overview

A **Postback API** is a server-to-server webhook that confirms a successful conversion (like a user registration). Your platform must make an HTTP request to our server when a tracked event occurs.

### Endpoint

```http
POST https://yourdomain.com/api/postback
Content-Type: application/json
```

### JSON Payload

The request body must be a JSON object containing the following fields:

- `code` (string, required): The unique tracking code generated by the publisher or traffic user. This is typically captured from the URL parameter when they land on your site.
- `uid` (string, required): The unique ID of the user on *your* system (e.g., the User ID or Phone Number from the game). This ensures duplicate conversions are ignored.
- `type` (string, required): The type of link being credited. Must be either `"publisher"` or `"traffic"`.
- `secret` (string, required): The secret postback key to authenticate your server. Default development secret is `your-postback-secret-key-for-dev` (or the value set in GlobalConfig).

### Example Request

```json
{
  "code": "A1B2C3D4E5",
  "uid": "user_987654321",
  "type": "traffic",
  "secret": "your-postback-secret-key-for-dev"
}
```

### Example cURL

```bash
curl -X POST https://yourdomain.com/api/postback \
  -H "Content-Type: application/json" \
  -d '{
    "code": "A1B2C3D4E5",
    "uid": "user_987654321",
    "type": "traffic",
    "secret": "your-postback-secret-key-for-dev"
  }'
```

### Expected Responses

**Success (200 OK):**
```json
{
  "success": true,
  "credited": 3.00
}
```

**Ignored / Duplicate / Invalid (200 OK with success: false):**
The API intentionally returns `200 OK` even for logical errors to prevent external systems from endlessly retrying.

```json
{
  "success": false,
  "reason": "duplicate" 
}
```
*Note: Possible error reasons include `invalid secret`, `link not found`, `link is inactive`, and `duplicate`.*
